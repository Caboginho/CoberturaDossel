# Checklist de Execução da Validação

Use este checklist para cada imagem analisada. Marque os itens durante a execução manual.

## Identificação

- [ ] Registrar `id_imagem`.
- [ ] Registrar `nome_arquivo`.
- [ ] Registrar `origem_imagem`.
- [ ] Registrar `licenca_ou_autorizacao`.
- [ ] Registrar `grupo_visual`.
- [ ] Registrar `condicao_ceu`.
- [ ] Registrar `tipo_ambiente`.

## Execução no Aplicativo

- [ ] Abrir o app.
- [ ] Criar nova análise.
- [ ] Informar nome padronizado da análise.
- [ ] Registrar observações iniciais, se houver.
- [ ] Importar imagem da galeria ou capturar com câmera.
- [ ] Confirmar que a imagem foi copiada para armazenamento interno.
- [ ] Gerar máscara automática.
- [ ] Registrar percentual de céu automático.
- [ ] Registrar percentual de dossel automático.
- [ ] Alternar visualizações para inspecionar imagem e máscara.
- [ ] Abrir editor de máscara.
- [ ] Corrigir erros visíveis.
- [ ] Registrar tempo aproximado de correção.
- [ ] Validar máscara final.
- [ ] Registrar percentual de céu final.
- [ ] Registrar percentual de dossel final.
- [ ] Registrar diferença percentual.
- [ ] Salvar análise.
- [ ] Exportar CSV.
- [ ] Exportar JSON.
- [ ] Registrar caminhos ou nomes dos arquivos exportados.

## Observações

- [ ] Registrar principais erros de segmentação.
- [ ] Registrar se a imagem original permaneceu inalterada.
- [ ] Registrar se a máscara automática permaneceu separada.
- [ ] Registrar se a máscara final foi criada como novo arquivo.
- [ ] Registrar qualquer travamento, mensagem confusa ou comportamento inesperado.
