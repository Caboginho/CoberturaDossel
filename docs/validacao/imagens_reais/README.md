# Imagens Reais

Esta pasta é reservada para imagens reais usadas na validação funcional local do MVP.

Não versione imagens reais neste repositório sem autorização explícita. Registre a origem, a licença ou a autorização de uso no modelo `docs/validacao/modelos/modelo_registro_validacao.csv`.

As imagens aqui devem ser usadas apenas para executar o fluxo do aplicativo. A imagem original deve permanecer preservada; o aplicativo deve gerar máscaras e resultados em arquivos separados.
