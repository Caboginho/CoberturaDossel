# Evidências Brutas

Esta pasta é reservada para evidências brutas da validação local, como capturas de tela, arquivos exportados temporários ou imagens de apoio ainda não revisadas.

Não versione evidências brutas quando elas contiverem imagens reais, dados sensíveis, material protegido por direitos autorais ou arquivos sem autorização de redistribuição.

Antes de publicar qualquer evidência, gere uma versão revisada e autorizada em `docs/validacao/evidencias/` ou registre apenas uma descrição textual no relatório de validação.
